# game

Game wrapper library internally using [cray](http://github.com/mswieboda/cray) ([RayLib](http://raylib.com) bindings to Crystal).

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     game:
       github: mswieboda/game
   ```

2. Run `shards install`

## Usage

```crystal
require "game"
```
